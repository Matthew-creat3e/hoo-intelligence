```

STORE/PROJECT: [NoReturn / TCB / HOO / Lead Engine / etc.]
BUILT:         [what was created or completed]
PUSHBACK:      [anything Matthew corrected or redirected]

NEXT FOCUS:    [highest priority for next session]
IMPROVE:       [what to do differently next time]
```
